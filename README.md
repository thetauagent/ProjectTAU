# ProjectTAU

ProjectTAU is the new scaffold for the animated TAU interface.

## Working idea
- dashboard-first shell
- living Office / presence scene
- persistent chat
- project builder harness
- custom workspaces with rearrangeable modules

## Planned repo shape
- `apps/web-shell` — the main web dashboard and workspace shell
- `apps/office` — the animated presence / Office layer
- `packages/ui` — shared UI primitives
- `packages/scene` — motion and scene helpers
- `packages/voice` — speak/listen helpers and audio state logic
- `packages/workspaces` — layout presets and module state
- `packages/agents` — instance and conversation abstractions
- `packages/shared` — shared types, state, and utilities

## Current status
This is only the initial scaffold. The actual framework choice can be finalized after the structure is approved.
